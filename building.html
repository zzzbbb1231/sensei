Building from source

Get the code

Check out the code: 

git clone git://sensei-ba.github.com/sensei.git

Build

SenseiBA builds using Apache Maven.

Now we can build the Sensei distribution, with: mvn clean install

Let's go to the sensei-ba directory and build it too: 

cd sensei-ba
mvn clean install

Code structure

The following is a quick overview of important directories in the sensei-ba repository:

  • assembly - contains metadata to build a tarball
  • config-example - SenseiBA examples
    • config-standalone - directory based indexing
    • config - zookeeper based segment management
  • ba-core - source code the real guts of SenseiBA
  • sensei-gateways - a collection of Sensei gateways
  • gazelle - compressed indexing structures
  • ba-hadoop - the stub project for the Sensei-BA Hadoop specific stuff
